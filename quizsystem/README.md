# Online Quiz System (Spring Boot)

## Features
- Add, View Quiz Questions
- REST API based
- MySQL Database Integration

## Run the Application
```bash
./mvnw spring-boot:run
```

## API Endpoints
- `GET /api/questions` - List all questions
- `POST /api/questions` - Add a new question
